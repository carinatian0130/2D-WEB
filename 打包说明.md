# 台本一键出视频工具包

## 启动方式

双击 `启动工具.bat`。本工具包已包含 Python 运行时，目标电脑不需要提前安装 Python 或 pip。

## 首次使用

1. 启动后会自动打开浏览器。
2. 进入"设置"页填写 OpenAI / 分镜 / 即梦相关 API Key。Base URL、模型 ID 以及拆解/分镜提示词已经在包内 `.env` 预置好，可直接使用，也可在"设置"页继续修改。
3. 用户自己的 API Key、修改后的配置、项目数据库、上传素材和生成视频都保存到当前 Windows 用户目录：`%APPDATA%\Script2Video`。

## 更新方式

以后更新程序时，只需要关闭工具，删除旧工具包或解压覆盖程序文件，再启动新版本即可。若忘记关闭旧版，`启动工具.bat` 会尝试先关闭后台旧版进程，再启动当前目录中的新版程序。
用户自己的项目、分镜图、生成视频、素材库和 API Key 等设置都保存在 `%APPDATA%\Script2Video`，不会因为替换工具包而丢失。
如果是从旧版本升级，新版本首次启动会自动把旧工具包目录里的 `database.db` 和 `uploads` 迁移到 `%APPDATA%\Script2Video`。

## 附属工具：导出旧版本数据.exe

如果你之前用的是把 `database.db` / `.env` / `uploads` 直接保存在程序根目录的旧版本，想把全部项目和素材带到本新版工具：

1. 把本工具包根目录的 `导出旧版本数据.exe` 复制到旧版工具的根目录（与旧版的 `database.db`、`uploads` 同一目录）。
2. 双击运行，会在该目录生成一个 `script2video-old-data-*.zip` 备份。
3. 回到本新版工具，进入"设置 → 数据迁移 → 导入数据备份"，选择刚才生成的 zip 即可。

该工具是独立 EXE，不依赖任何 Python 环境，双击即可运行。

## 隐私与打包排除

本包不包含任何 API Key（OpenAI / 分镜 / 即梦 / 火山方舟 等 `*_API_KEY` 字段均已从随包 `.env` 中剔除），也不包含 `.env.example`、项目数据、分镜图、生成视频、缓存文件以及 `QDD幻灯片自动创建工具` 文件夹。
本包会携带已脱敏的初始化素材库数据库，仅保留场景库、人物库记录，并复制 `uploads/scenes` 与 `uploads/characters` 中的图片素材。首次启动且用户目录没有数据库时，这些内容会作为初始化数据迁移到 `%APPDATA%\Script2Video`。
本包会携带一份脱敏的 `.env`，预置 Base URL、模型 ID 与默认提示词等非敏感配置，方便他人开箱即用。运行时优先读取 `%APPDATA%\Script2Video\.env` 中用户自己的配置，包内 `.env` 仅作为默认值。

如需迁移给其他电脑，只发送本工具包即可；不要发送 `%APPDATA%\Script2Video`，除非明确需要连同本机用户数据一起迁移。
