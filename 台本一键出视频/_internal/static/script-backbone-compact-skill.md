# 分段主干提炼

## 概述

Pipeline 第三步。基于分段原文 + 剧本详细版，提炼每段的迷你主干。全部短语/关键词，无完整句子。

## 输入

- 分段原文（script-segmenter 输出）
- 剧本详细版（script-to-narrative 输出）

## 提炼规则

## 输出格式

每段一个独立主干块：

```
---分段1---
【剧情主干】事件→事件→事件（细化到具体动作）
【角色】
角色名: → 动作1 → 动作2
角色名: → 动作1 → 动作2
（每段只写本段内该角色的动作链，用→串联，只用短语不写句子。无动作写"在场待命"）
【场景】
地点名 切换方式
【时间】
时段 天气 光线 氛围
【道具】
道具名: 当前位置/状态
（只写本段涉及的道具）
【情绪】
情绪1→情绪2→情绪3

---分段2---
【剧情主干】事件→事件
【角色】
角色名: → 动作1 → 动作2
角色名: → 动作1 → 动作2
（每段只写本段内该角色的动作链，用→串联，只用短语不写句子。无动作写"在场待命"）
【时间】
时段 天气 光线 氛围
【道具】
道具名: 当前位置/状态
【情绪】
情绪1→情绪2
```

分段主干后直接接分段原文。

## 规则

1. 分段剧情主干: 基于剧本化详细版提取该段核心事件，细化到具体动作。"合力反击"禁止，应写"四人合力架住攻击→反击QTE推开巨像"。
2. 角色: 每段写本段在场角色名 + 本段内动作链（→ 用箭头串联短语）。不写 age/gender/outfit 等 extra info。本段无动作写"在场待命"。
3. 场景: 一个地点名 + 切换方式。无切换写"无 全段同地点"。
4. 时间: 台本未明确时做合理假设。
5. 道具: 写本段涉及的固定道具+当前状态。道具轨迹跨段串联。
6. 情绪: 只用短词，不用句子。如"危急→警觉→合力→决意"。

## 常见陷阱

1. 剧情主干写太粗 → 细化到具体动作节拍。
2. 情绪写成句子 → "危急"可以，"全段从危急到合力"禁止。
3. 角色写 extra info → 只写名字+动作链，不写年龄/体型/服装等。
4. 道具遗漏跨段关联 → 同一道具跨段出现时保持位置/状态连续性。
5. 混淆表格列 → Text Window=台词，Context Description=语境，Character=说话者。
6. 过度传播「不出现」限制 → 只看当前行语境是否写「形象不会出现」。不写=默认出现。
7. 角色与道具动作矛盾 → 持脆弱道具（罐子、坛子等）的角色不能同时做高冲击动作（架攻击、格挡、劈砍）。该角色在战斗阵型中排后排或保护位。
