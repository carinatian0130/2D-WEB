# 参考分镜与空间图

## 参考分镜匹配

当用户同时提供参考分镜图和对白/场景文本时，生成导演预演分镜提示词，并把参考图当作强视觉依据，不当作宽泛灵感。

尽量匹配参考图的格数、画幅比例、分格布局、阅读顺序、机位、景别、主体尺度、前中后景层次、人物调度、道具锚点、视线轴和情绪节奏。

不要机械复制成互不相关的静帧。输出必须像一个连续导演过的场景。面板之间要通过可见桥接相连，例如同一桌面轴线、同一道具位置、视线延续、手部动作延续、身体重心变化、摄影机推拉平移方向、前景物体穿过画面，或情绪因果。

参考图可作为角色外貌证据。锁定每个角色的身份、发型、服装轮廓、体型、武器/道具和可读剪影。如果用户指定【图一】、【图二】这类身份，在提示词中使用该标签并保持稳定。

## 四格参考分镜模式

除非用户另有要求，否则使用以下模式：

1. 第一格：宽景或中宽景动态建立，复现参考图的房间/桌面/角色布局、冲突触发点、主要道具锚点、光线方向和第一镜头方向。
2. 第二格：反应或冲突，延续同一空间轴线但明显改变尺度/角度，常用越肩或侧轴，并用前景肩膀、身体或道具连接剪辑。
3. 第三格：视觉转折特写，匹配参考图的插入/特写逻辑，使用关键手、脸、道具、武器或冲击点；这是最强的尺度变化，但仍保留桌面、地面或光线连续性。
4. 第四格：情绪落点或决定，回到可读关系构图，展示关系距离或权力平衡的变化。

## 平面调度图

当场景空间关系重要时，在同一提示词中加入一个平面调度图小插图：顶视房间或环境布局、角色位置、关键道具位置、入口/出口、光线方向、摄影机路径、人物运动路径和视线轴。

只有用户要求位置标签时，才使用简短实用标签，例如第一格到第四格、【图1】、【图2】、桌、罐、钱、门、窗、摄影机箭头和简单符号。否则尽量减少可读标签，不写对白。禁止完整对白、字幕、界面元素、水印或长说明文字。

## 室内空间四维图

当所选场景为室内时，在单张图提示词中加入一个额外的空间四维图面板或小插图，用于提升图生视频的摄影机和角色定位准确性。室外场景不加入空间四维图。

空间图必须保持黑白铅笔分镜风格，可作为宽横向小插图、边栏、下方小图或角落调度图放在分镜旁边：俯视室内平面、主要家具/障碍物、角色位置、入口/出口、窗、关键道具、必要的编号或符号位置标记，以及少量不可读的图例式箭头痕迹。

使用俯视平面、中央桌周围角色、可见环境锚点、蓝色虚线摄影机路径、红色运动箭头、绿色构图/视线轴、必要时橙色光线箭头，以及紫色/黄色情绪或重点标记。

室内空间图必须描述四个维度：

1. 平面位置：顶视室内布局，包括入口、出口、主要家具、墙、窗、门洞、角色位置和关键道具。
2. 纵深轴：前景/中景/背景关系，人物到门、窗、桌、柜台、楼梯或主要障碍物的距离。
3. 高度轴：地面高度、桌面/柜台/台阶高度、坐姿/站姿角色高度、栏杆/灯具/窗台高度。
4. 时间/运动轴：角色运动路径、摄影机运动路线、视线/关系轴，以及关键道具在各面板中的位置变化。

室内场景的空间图辅助分镜，不替代分镜。它必须服务十六比九可切镜头：每个主分镜格仍按十六比九有效镜头框设计，且所有主分镜格必须等宽、等高、同一比例、同一视觉尺度。空间图、标注区或留白只能用于吸收整张图比例，不能插入主分镜网格造成主格大小不一，也不能让主分镜格被挤压或拉伸。

描述空间图时，用几何形状画物件，不用模糊标签：圆桌画成圆或椭圆，柜台/桌柜画成长方形，楼梯画成阶梯带，门画成带转轴的矩形，窗画成窄矩形，路径画成箭头。空间图必须和分镜道具一致，分镜道具也必须和空间图一致。
